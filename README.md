# Steam


## Frontend

### Technologies
- ReactJS
- Typescript
- CRA

### How to build
    cd engn_steam/client
    yarn install

### How to run
    cd engn_steam/client
    yarn start

## Backend

### Technologies
- Postgres
- Typescript
- NodeJS
### How to build
    // run postgres

    cd engn/server
    yarn install
    yarn typeorm migration:run

### How to run
    cd engn/server
    yarn dev
